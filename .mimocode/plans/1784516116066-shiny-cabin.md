## Task 4: Add MCP Authentication

### Current State
- No authentication implemented in MCP server (`src/TiaAgent.McpHost/Program.cs`)
- `appsettings.json` has `"RequireAuthentication": true` but not enforced
- Security model specifies: "ephemeral bearer token per session" (§14.1)
- `ErrorMapper` has `MCP_AUTHENTICATION_FAILED` error code defined

### Implementation Plan
1. **Generate ephemeral session token** when MCP server starts
2. **Add authentication middleware** to ASP.NET Core pipeline
3. **Pass token to OpenCode** via configuration
4. **Validate token on every MCP request**
5. **Return 401 Unauthorized** for missing/invalid tokens

### Files to Modify
- `src/TiaAgent.McpHost/Program.cs` — Add auth middleware
- `src/TiaAgent.Mcp/Auth/` — Create auth classes
  - `McpAuthenticationMiddleware.cs` — Validate bearer token
  - `SessionTokenProvider.cs` — Generate/manage tokens
- `config/appsettings.json` — Token configuration

### Implementation Details

**Token Generation:**
```csharp
public class SessionTokenProvider
{
    private readonly string _token;
    
    public SessionTokenProvider()
    {
        _token = GenerateSecureToken(); // 32-byte random
    }
    
    public string GetToken() => _token;
    
    private static string GenerateSecureToken()
    {
        var bytes = new byte[32];
        using (var rng = RandomNumberGenerator.Create())
        {
            rng.GetBytes(bytes);
        }
        return Convert.ToBase64String(bytes);
    }
}
```

**Authentication Middleware:**
```csharp
public class McpAuthenticationMiddleware
{
    private readonly RequestDelegate _next;
    private readonly SessionTokenProvider _tokenProvider;
    
    public async Task InvokeAsync(HttpContext context)
    {
        if (!context.Request.Headers.TryGetValue("Authorization", out var authHeader) ||
            !authHeader.ToString().StartsWith("Bearer "))
        {
            context.Response.StatusCode = 401;
            return;
        }
        
        var token = authHeader.ToString().Substring("Bearer ".Length);
        if (token != _tokenProvider.GetToken())
        {
            context.Response.StatusCode = 401;
            return;
        }
        
        await _next(context);
    }
}
```

**Program.cs Changes:**
```csharp
// Add before app.MapMcp()
var tokenProvider = new SessionTokenProvider();
app.UseMiddleware<McpAuthenticationMiddleware>();

// Log token for OpenCode configuration (without exposing in logs)
Console.WriteLine($"MCP Server Token: {tokenProvider.GetToken()}");
```

### Security Considerations
- Token must be generated fresh on each MCP server restart
- Token must not be logged (security model §10)
- Token must be passed to OpenCode via secure configuration
- Token must be validated on every request
